# PostgreSQL Tutorial

This repository contains my notes and SQL scripts from following a PostgreSQL tutorial on YouTube. Everything here is written as a follow-along of the video — the code and concepts are based on the tutorial's teaching, not original work of my own.

The tutorial is split into three parts, each building on the last.

## Part 1 — Database Fundamentals
Covers the basics of setting up and structuring a database:
- Creating and managing **databases**
- Working with **schemas**
- Creating **tables**
- Understanding PostgreSQL **data types**

## Part 2 — Core SQL Concepts
Covers the everyday SQL operations used to work with data:
- `INSERT INTO` — adding records
- `SELECT` — querying data
- `LIKE` / `ILIKE` — pattern matching (case-sensitive and case-insensitive)
- `ORDER BY` — sorting results
- `GROUP BY` — grouping records
- `UPDATE` — modifying existing records
- `NULL` / `NOT NULL` — handling missing or required data
- `LIMIT` / `OFFSET` — paginating results
- `DELETE` — removing records
- `RETURNING` — getting back data affected by a query

## Part 3 — Relationships & Joins
Covers how to model and query relationships between tables:
- `RESET` — resetting sequences/state
- Seeding data and relationships
- **Many-to-many** relationships
- **One-to-many** relationships
- `INNER JOIN`
- `LEFT JOIN`
- `COUNT`
- `DISTINCT`
- Table **aliases**
- Aggregate functions
